# Track — Pilot

## Цель
Полигон для проверки нового подхода: один Claude-оркестратор владеет итерацией целиком, делегирует параллельную работу под-агентам, не нужны kanban-диспатч и verifying-цикл.

## Scope
- Мелкие добавочные задачи в kanban-приложении.
- Цель — не сами фичи, а проверка orchestrator-команды `/iterate`.

## Заметки
Если оркестратор справится с итерацией 001 — оформляем подход в шаблоны и переносим в `templates/.claude/commands/`.
