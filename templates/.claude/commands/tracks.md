---
description: "Список всех активных треков с количеством тасков по статусам."
allowed-tools: Read, Glob
---

Покажи все активные треки.

Шаги:
1. Найди все папки `.workflow/tracks/*/`. Если пусто — скажи "Нет треков. Создай через /new-track" и закончи.
2. Для каждого трека:
   - Прочитай `README.md`, достань цель (первый параграф после `## Цель`).
   - Прочитай все `tasks/T*.md`, посчитай таски по статусам.
3. Выведи компактным списком:
   ```
   <slug> — <цель одной строкой>
     todo: N · in-progress: N · review: N · blocked: N · done: N · total: N
   ```
4. Если задана активная итерация — намекни в конце: «Подробности по треку: /track <slug>; по итерации: /iter».

Не редактируй ничего.
